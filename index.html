05 · CAPABILITIES

System Features

A full suite of smart city traffic tools.

🧠

AI Adaptive Control

Real-time congestion analysis allocates green time proportionally to lane density.

🚨

Emergency Priority

One-click emergency override clears a path instantly, resuming auto control after 15s.

🕹

Manual Override

Operators can force-green any direction while the system pauses its cycle.

📊

Live Analytics

Real-time Chart.js graphs track density trends and green-time distribution.

⏱

Countdown Timers

Each traffic light shows a live countdown so drivers can anticipate signal changes.

🚗

Car Simulation

Animated vehicles spawn and queue based on your density sliders, halting at red lights.

🌐

Responsive Design

Fully functional on desktop and tablet. Pure HTML/CSS/JS — no server required.

⚡

Zero Dependencies*

Only Chart.js (CDN) is external. Works offline after first load.
